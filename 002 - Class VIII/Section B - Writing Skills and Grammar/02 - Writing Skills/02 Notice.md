# Notice Writing

## 📌 What is a Notice?
A **notice** is a formal written communication intended to announce information, instructions, or events to a specific group of people. In schools, notices inform students about competitions, camps, lost/found items, or sudden holidays.

## 📝 Format / Structure (Word Limit: 50 words)

A notice must always be enclosed in a **BOX**.

```text
+-----------------------------------------------------------------------------+
|                        NAME OF THE INSTITUTION/SCHOOL                       |
|                                                                             |
|                                  NOTICE                                     |
|                                                                             |
| Date (e.g., 14 May 2026)                                                    |
|                                                                             |
|                             HEADING / TITLE                                 |
|                                                                             |
| Body of the Notice: (Target audience, what the event is, date, time, venue, |
| and whom to contact for further details. Keep it brief and to the point.)   |
|                                                                             |
| Signature                                                                   |
| Name                                                                        |
| Designation                                                                 |
+-----------------------------------------------------------------------------+
```

## ⚖️ CBSE Marking Scheme (Typically 4-5 Marks)
- **Format**: 1 Mark (Box, Name of Institution, "NOTICE", Date, Heading, Name/Designation)
- **Content**: 2 Marks (What, When, Where, Whom to contact)
- **Expression**: 1 Mark (Grammar, Spelling, Formal Tone)

## 💡 Language Tips & Key Rules
- **Tone**: Formal, objective, and polite. Avoid first-person pronouns (I, me, my, we).
- **Passive Voice**: Use the passive voice to maintain formality (e.g., *It is hereby informed that...*).
- **The 5 Ws**: Ensure your notice answers: *What* is happening? *Where* is it happening? *When* is it happening? *Who* can participate? *Whom* to contact?
- **Word Limit**: Do not exceed 50 words for the main body.

## ⚠️ Common Mistakes to Avoid
- **No Box**: A notice written without a bounding box will automatically lose marks.
- **Informal language**: Starting with "Hello students, I want to tell you..." is strictly incorrect.
- **Vague details**: Forgetting to mention the specific time or venue of the event.

## 🌟 Model Example

**Q: You are the Secretary of the Science Club of Delhi Public School. Write a notice in about 50 words informing students of Classes VIII to X about an upcoming Science Exhibition.**

+-----------------------------------------------------------------------------+
|                            DELHI PUBLIC SCHOOL                              |
|                                                                             |
|                                    NOTICE                                   |
|                                                                             |
| 12 August 2026                                                              |
|                                                                             |
|                             SCIENCE EXHIBITION                              |
|                                                                             |
| This is to inform all students of Classes VIII to X that the Science Club   |
| is organizing a Science Exhibition on 25 August 2026 in the school          |
| auditorium from 10:00 AM to 1:00 PM. Students interested in presenting      |
| working models or projects must register their names with the undersigned   |
| by 18 August 2026.                                                          |
|                                                                             |
| (Signature)                                                                 |
| Aryan Mehta                                                                 |
| Secretary, Science Club                                                     |
+-----------------------------------------------------------------------------+

## ✍️ Practice Prompts

1. **Found:** You are Neha/Nikhil of Class VIII. You have found a smart watch in the school library. Write a notice for the school notice board asking the owner to claim it with proper identification.
2. **Event:** Your school is organizing an excursion to the local planetarium. As the Head Boy/Girl, write a notice informing students of Class VIII about the trip, mentioning the fee and date.
