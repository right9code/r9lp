# Notice Writing

## 📌 What is a Notice?
A **notice** is a formal, concise piece of writing used by institutions, schools, or organizations to announce events, issue instructions, or make appeals to a targeted audience. It must be brief, factual, and to the point.

## 📝 Format / Structure (Word Limit: 50 words)

A notice MUST be enclosed in a **BOX**.

```text
+-----------------------------------------------------------------------------+
|                        NAME OF THE INSTITUTION/SCHOOL                       |
|                                                                             |
|                                  NOTICE                                     |
|                                                                             |
| Date (e.g., 14 May 2026)                                                    |
|                                                                             |
|                             HEADING / TITLE                                 |
|                                                                             |
| Body of the Notice: (Target audience, what the event is, date, time, venue, |
| and whom to contact for further details. Keep it brief and to the point.)   |
|                                                                             |
| Signature                                                                   |
| Name                                                                        |
| Designation                                                                 |
+-----------------------------------------------------------------------------+
```

## ⚖️ CBSE Marking Scheme (Typically 4-5 Marks)
- **Format**: 1 Mark (Box, Name of Institution, "NOTICE", Date, Heading, Name/Designation)
- **Content**: 2 Marks (What, When, Where, Whom to contact)
- **Expression**: 1 Mark (Grammar, Spelling, Formal Tone)

## 💡 Language Tips & Key Rules
- **Tone**: Formal and objective. Avoid first-person pronouns (I, we) unless used in the designation (e.g., "The undersigned").
- **Passive Voice**: Prefer passive structures (e.g., *It is hereby notified that...* or *Students are informed that...*).
- **The 5 Ws**: Ensure your notice answers: *What* is the event? *Where* is the venue? *When* is the date and time? *Who* is eligible? *Whom* to contact?
- **Word Limit**: Strictly adhere to the 50-word limit for the body.

## ⚠️ Common Mistakes to Avoid
- **Forgetting the Box**: This is a guaranteed deduction of marks.
- **Informal language**: Do not use conversational greetings like "Hello friends".
- **Missing deadlines**: If an event requires registration, always mention a "last date to register".

## 🌟 Model Example

**Q: You are the Cultural Secretary of Bal Bharati Public School, Delhi. Your school is organizing an Inter-House Dramatics Competition. Write a notice in about 50 words inviting names from students of Classes IX and X.**

+-----------------------------------------------------------------------------+
|                        BAL BHARATI PUBLIC SCHOOL, DELHI                     |
|                                                                             |
|                                    NOTICE                                   |
|                                                                             |
| 20 August 2026                                                              |
|                                                                             |
|                       INTER-HOUSE DRAMATICS COMPETITION                     |
|                                                                             |
| This is to inform all students of Classes IX and X that the school is       |
| organizing an Inter-House Dramatics Competition on 5 September 2026 in the  |
| school auditorium from 10:00 AM onwards. The theme for the play is "Unity   |
| in Diversity". Interested students must register their names with their     |
| respective House Captains latest by 28 August 2026.                         |
|                                                                             |
| (Signature)                                                                 |
| Kavita Sharma                                                               |
| Cultural Secretary                                                          |
+-----------------------------------------------------------------------------+

## ✍️ Practice Prompts

1. **Tour/Camp:** You are the Sports Captain. Write a notice informing students of Class IX about a 3-day Trekking Camp to Shimla. Mention dates, fees, and the last date for submitting consent forms.
2. **Found:** You found a silver wristwatch in the school computer lab. Write a notice for the school notice board asking the rightful owner to claim it by providing necessary identification details.
