# Poster Writing

## 📌 What is a Poster?
A **poster** is a large notice or placard announcing or advertising something. It combines visual appeal with a strong written message. Posters are created to create social awareness about an issue (like blood donation or saving water) or to advertise an event (like a school fete).

## 📝 Format / Structure (Word Limit: 50 words)

A poster is a visual medium. It MUST be drawn inside a **BOX**.

A good poster includes:
1. **Layout / Visuals**: 
   - A catchy title in bold/capital letters.
   - Simple sketches, stick figures, or visual symbols.
   - Varied font sizes and styles to make it attractive.
2. **Content**:
   - The theme or subject of the poster.
   - A catchy slogan or jingle.
   - Essential details (Date, Time, Venue - if it's for an event).
   - "Issued in public interest by: [Name/Organization]" at the bottom.

## ⚖️ CBSE Marking Scheme (Typically 3-4 Marks)
- **Format**: 1 Mark (Box, layout, visual appeal)
- **Content**: 1-2 Marks (Theme, slogan, necessary details)
- **Expression**: 1 Mark (Creativity, accuracy of language)

## 💡 Language Tips & Key Rules
- **Less is More**: Do not write long paragraphs. Use short phrases, bullet points, and slogans.
- **Use Action Verbs**: "Donate!", "Save!", "Join us!"
- **Visuals matter**: You do not need to be a great artist. Simple, neat sketches or even just creative use of fonts (bubble letters, bold letters) is enough to score full marks.

## ⚠️ Common Mistakes to Avoid
- Forgetting to draw the bounding box.
- Forgetting to write the issuing authority ("Issued by...") at the bottom.
- Writing full, long sentences instead of catchy phrases.

## 🌟 Model Example Layout

**Q: Design a poster in not more than 50 words to create awareness about the need to save trees.**

```text
+-----------------------------------------------------------------------------+
|                                                                             |
|                           SAVE TREES, SAVE LIFE!                            |
|                                                                             |
|                    [Sketch of a tree with a crying face]                    |
|                                                                             |
|        TREES ARE OUR BEST FRIENDS!                                          |
|                                                                             |
|        * They give us Oxygen.                                               |
|        * They bring Rain.                                                   |
|        * They prevent Soil Erosion.                                         |
|                                                                             |
|             DON'T CUT THEM! PLANT MORE SAPLINGS TODAY!                      |
|                                                                             |
|                                                                             |
|                                                                             |
|   Issued in public interest by:                                             |
|   The Eco Club, Delhi Public School                                         |
+-----------------------------------------------------------------------------+
```
