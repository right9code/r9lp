# Notice Writing

## 📌 What is a Notice?
A **notice** is a formal means of communication. Its purpose is to announce or display information to a specific group of people. Notices are generally pinned up on school or public notice boards. They cover events, competitions, lost and found items, inaugurations, or general instructions.

## 📝 Format / Structure (Word Limit: 50 words)

A notice must always be enclosed in a **BOX**.

```text
+-----------------------------------------------------------------------------+
|                        NAME OF THE INSTITUTION/SCHOOL                       |
|                                                                             |
|                                  NOTICE                                     |
|                                                                             |
| Date (e.g., 14 May 2026)                                                    |
|                                                                             |
|                             HEADING / TITLE                                 |
|                                                                             |
| Body of the Notice: (Target audience, what the event is, date, time, venue, |
| and whom to contact for further details. Keep it brief and to the point.)   |
|                                                                             |
| Signature                                                                   |
| Name                                                                        |
| Designation                                                                 |
+-----------------------------------------------------------------------------+
```

## ⚖️ CBSE Marking Scheme (Typically 4-5 Marks)
- **Format**: 1 Mark (Box, Name of Institution, "NOTICE", Date, Heading, Name/Designation)
- **Content**: 2 Marks (What, When, Where, Whom to contact)
- **Expression**: 1 Mark (Grammar, Spelling, Tone)

## 💡 Language Tips & Key Rules
- **Tone**: Formal and objective. Do NOT use first-person pronouns like "I" or "We" (unless quoting a specific authority). Use passive voice where possible.
- **The 5 Ws**: Ensure your notice answers: *What* is happening? *Where* is it happening? *When* is it happening? *Who* can participate? *Whom* to contact?
- **Word Limit**: Do not exceed 50 words in the body.
- **MUST BE IN A BOX**: The notice will lose marks if it is not drawn inside a neat box.

## ⚠️ Common Mistakes to Avoid
- Forgetting to draw the bounding box.
- Using informal language or personal pronouns ("I am writing this notice to tell you...").
- Missing out on crucial details like the date, time, or venue of the event.
- Writing the date in the wrong format (e.g., Avoid 14/05/2026. Use 14 May 2026).

## 🌟 Model Example

**Q: You are the Head Boy/Girl of St. Mary's School, Delhi. Your school is organizing a blood donation camp. Write a notice in about 50 words informing students of classes IX to XII about the camp and urging them to encourage their parents to donate.**

+-----------------------------------------------------------------------------+
|                            ST. MARY'S SCHOOL, DELHI                         |
|                                                                             |
|                                    NOTICE                                   |
|                                                                             |
| 10 October 2026                                                             |
|                                                                             |
|                            BLOOD DONATION CAMP                              |
|                                                                             |
| This is to inform all students of Classes IX to XII that the school, in     |
| collaboration with the Red Cross Society, is organizing a Blood Donation    |
| Camp on 15 October 2026, from 9:00 AM to 2:00 PM in the school auditorium.  |
| Students are requested to encourage their parents and relatives to come     |
| forward and donate blood for a noble cause. For further details, contact    |
| the undersigned.                                                            |
|                                                                             |
| (Signature)                                                                 |
| Rahul Sharma                                                                |
| Head Boy                                                                    |
+-----------------------------------------------------------------------------+

## ✍️ Practice Prompts

1. **Lost & Found:** You are Anjali/Amit of Class VII. You have lost your pencil box in the school playground. Write a notice for the school notice board giving details of the lost item and offering a suitable reward.
2. **Event:** Your school is organizing an Inter-House Debate Competition. As the Cultural Secretary, write a notice informing students about the event and inviting names for participation.
