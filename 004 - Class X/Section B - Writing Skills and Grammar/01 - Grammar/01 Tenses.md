# TENSES

Tenses denote the time of action in a sentence.

  
There are three main tenses:

1. Present Tense: Describes actions happening now or regularly.
    
2. Past Tense: Describes actions that already happened.
    
3. Future Tense: Describes actions that will happen later.
    

  

Each tense have four sub-tenses:

1. Indefinite/Simple.
    
2. Continuous.
    
3. Perfect.
    
4. Perfect Continuous.
    

  

So in total there are 12 tenses.

## 1. Present Indefinite Tense

  

The Present Indefinite Tense (also known as the Simple Present Tense) is used to express habitual actions, universal truths, fixed schedules, and general facts.

  

Base Rule (Affirmative Sentences) :

|   |
|---|
|Singular Subject (III) + Verb (I) + s/es + Object  <br>Other Subject + Verb (I) + Object<br><br>Add -es when the verb ends in: -ch, -sh, -s, -x, or -z (e.g., Watch → Watches).<br><br>Add -s for most other verbs (e.g., Eat → Eats).|

Examples:

- She eats an orange. (Habitual Action)
    
- I eat an apple every day. (Habitual Action)
    
- The Sun rises in the morning. (Universal Truth)
    
- The stars shine at night. (Universal Truth)
    
- The bell rings at 8:50 AM. (Fixed Schedule)
    
- Our classes start at 9 AM. (Fixed Schedule)
    
- Fish swims in water. (General Fact)
    
- Lions live in the forest. (General Fact)  
      
    

Other Sentences Rules :

- Negative Sentences
    

|   |
|---|
|Singular Subject (III) + does not+ Verb (I) + Object  <br>Other Subject + do not + Verb (I) + Object|

For sentences with Never:

|   |
|---|
|Singular Subject (III) + never + Verb (I) + s/es + Object  <br>Other Subject + never + Verb (I) + Object|

- Interrogative Sentences 
    

|   |
|---|
|(W/H words) + Does + Singular Subject (III) + Verb (I) + Object + ?  <br>(W/H words) + Do + Other Subject + Verb (I) + Object + ?|

W/H words are interrogative pronouns that are used to ask questions like what, when, why, how, etc.

- Negative Interrogative Sentences
    

|   |
|---|
|(W/H words) + Does + Singular Subject (III) + not + Verb (I) + Object + ?  <br>(W/H words) + Do + Other Subject + Verb (I) + not +Object + ?|

## 2. Present Continuous Tense

  

The Present Continuous Tense (also known as the Present Progressive) is used to describe actions that are happening right now, temporary situations, or planned future events.

  

Base Rule (Affirmative Sentences) :

|   |
|---|
|Singular Subject (III) + is + Verb (I) + ing + Object  <br>Other Subject + are + Verb (I) + ing + Object  <br>I + am + Verb (I) + ing + Object|

Examples:

- She is eating an orange. (Action happening now)
    
- I am eating an apple right now. (Action happening now)
    
- The Sun is rising behind the clouds. (Temporary action in progress)
    
- The stars are shining brightly tonight. (Current state/observation)
    
- The bell is ringing for the next period. (Action happening now)
    
- Our classes are starting late this week. (Temporary situation)
    
- The fish is swimming in the bowl. (Action happening now)
    
- Lions are living in protected reserves these days. (Temporary/Changing situation)
    

Other Sentences Rules :

- Negative Sentences
    

|   |
|---|
|Singular Subject (III) + is not + Verb (I) + ing + Object  <br>Other Subject + are not + Verb (I) + ing + Object  <br>I + am not + Verb (I) + ing + Object|

For sentences with Never:

|   |
|---|
|Singular Subject (III) + is never + Verb (I) + ing + Object  <br>Other Subject + are never + Verb (I) + ing + Object  <br>I + am never + Verb (I) + ing + Object|

- Interrogative Sentences 
    

|   |
|---|
|(W/H words) + Is + Singular Subject (III) + Verb (I) + ing + Object + ?  <br>(W/H words) + Are + Other Subject + Verb (I) + ing + Object + ?  <br>(W/H words) + Am + I + Verb (I) + ing + Object + ?|

- Negative Interrogative Sentences
    

|   |
|---|
|(W/H words) + Is + Singular Subject (III) + not + Verb (I) + ing + Object + ?  <br>(W/H words) + Are + Other Subject + not + Verb (I) + ing + Object + ?|

## 3. Present Perfect Tense

  

The Present Perfect Tense is used to describe actions that have been completed recently, or actions where the specific time of completion is not as important as the result in the present.

  

Base Rule (Affirmative Sentences) :

|   |
|---|
|Singular Subject (III) + has + Verb (III) + Object  <br>Other Subject + have + Verb (III) + Object|

Examples:

- She has eaten her orange. (Recently completed action)
    
- I have eaten my breakfast already. (Action finished in the past with present relevance)
    
- The Sun has risen high in the sky. (Result of a completed action)
    
- The stars have appeared in the sky. (Completed action)
    
- The bell has just rung. (Immediate past)
    
- Our classes have started for the new term. (Completed action)
    
- The fish has swum to the bottom of the tank. (Action finished)
    
- The lions have lived in this area for years. (Action starting in the past and continuing to now)
    

Other Sentences Rules :

- Negative Sentences
    

|   |
|---|
|Singular Subject (III) + has not + Verb (III) + Object  <br>Other Subject + have not + Verb (III) + Object|

For sentences with Never:

|   |
|---|
|Singular Subject (III) + has never + Verb (III) + Object  <br>Other Subject + have never + Verb (III) + Object|

- Interrogative Sentences 
    

|   |
|---|
|(W/H words) + Has + Singular Subject (III) + Verb (III) + Object + ?  <br>(W/H words) + Have + Other Subject + Verb (III) + Object + ?|

- Negative Interrogative Sentences
    

|   |
|---|
|(W/H words) + Has + Singular Subject (III) + not + Verb (III) + Object + ?  <br>(W/H words) + Have + Other Subject + not + Verb (III) + Object + ?|

## 4. Present Perfect Continuous Tense

The Present Perfect Continuous Tense is used to describe actions that started in the past and are still continuing in the present. It emphasizes the duration of the activity.

  

Base Rule (Affirmative Sentences) :

|   |
|---|
|Singular Subject (III) + has been + Verb (I) + ing + Object + (since/for) + Time  <br>Other Subject + have been + Verb (I) + ing + Object + (since/for) + Time<br><br>Use for to describe a duration (e.g., for two hours), and use since to describe a starting point (e.g., since 2 PM).|

Examples:

- She has been eating healthy food for a month. (Action continuing over time)
    
- I have been eating apples since I was a child. (Long-term habit continuing now)
    
- The Sun has been rising earlier each day this week. (Continuing trend)
    
- The stars have been shining all night. (Duration of action)
    
- The bell has been ringing for two minutes straight. (Ongoing action)
    
- Our classes have been starting on time lately. (Recent continuous behavior)
    
- The fish has been swimming near the surface all morning. (Duration)
    
- Lions have been living in this forest for centuries. (Long-term continuous state)
    

Other Sentences Rules :

- Negative Sentences
    

|   |
|---|
|Singular Subject (III) + has not been + Verb (I) + ing + Object + (since/for) + Time  <br>Other Subject + have not been + Verb (I) + ing + Object + (since/for) + Time|

For sentences with Never:

|   |
|---|
|Singular Subject (III) + has never been + Verb (I) + ing + Object + (since/for) + Time  <br>Other Subject + have never been + Verb (I) + ing + Object + (since/for) + Time|

- Interrogative Sentences 
    

|   |
|---|
|(W/H words) + Has + Singular Subject (III) + been + Verb (I) + ing + Object + (since/for) + Time + ?  <br>(W/H words) + Have + Other Subject + been + Verb (I) + ing + Object + (since/for) + Time + ?|

- Negative Interrogative Sentences
    

|   |
|---|
|(W/H words) + Has + Singular Subject (III) + not been + Verb (I) + ing + Object + (since/for) + Time + ?  <br>(W/H words) + Have + Other Subject + not been + Verb (I) + ing + Object + (since/for) + Time + ?|

## 5. Past Indefinite Tense

  

The Past Indefinite Tense (also known as the Simple Past Tense) is used to express actions that were completed in the past at a specific time, past habits, or historical facts.

  

Base Rule (Affirmative Sentences) :

|   |
|---|
|All Subjects + Verb (II) + Object<br><br>Regular Verbs: Formed by adding -ed or -d to the base form (e.g., Walk → Walked).<br><br>Irregular Verbs: These change their spelling entirely and do not follow a standard pattern (e.g., Go → Went, See → Saw).|

Examples:

- She ate an orange yesterday. (Completed Action)
    
- I ate an apple every day during my childhood. (Past Habit)
    
- The Sun rose at 5:45 AM this morning. (Completed Event)
    
- The stars shone brightly last night. (Completed Action)
    
- The bell rang at 8:50 AM. (Specific Time in the Past)
    
- Our classes started last week. (Completed Schedule)
    
- The fish swam across the pond. (Completed Action)
    
- Lions lived in this valley many years ago. (Historical Fact)
    

  

  

Other Sentences Rules :

- Negative Sentences
    

|   |
|---|
|All Subjects + did not + Verb (I) + Object|

For sentences with Never:

|   |
|---|
|All Subjects + never + Verb (II) + Object|

- Interrogative Sentences 
    

|   |
|---|
|(W/H words) + Did + All Subjects + Verb (I) + Object + ?|

- Negative Interrogative Sentences
    

|   |
|---|
|(W/H words) + Did + All Subjects + not + Verb (I) + Object + ?|

## 6. Past Continuous Tense

  

The Past Continuous Tense (also known as the Past Progressive) is used to describe actions that were ongoing at a specific point in the past or were interrupted by another action.

  

Base Rule (Affirmative Sentences) :

|   |
|---|
|Singular Subject (III) + was + Verb (I) + ing + Object  <br>Other Subject + were + Verb (I) + ing + Object  <br>I + was + Verb (I) + ing + Object|

Examples:

- She was eating an orange when I arrived. (Action in progress at a past point)
    
- I was eating an apple while reading a book. (Parallel past actions)
    
- The Sun was rising as we reached the summit. (Ongoing past action)
    
- The stars were shining when the power went out. (Background state)
    
- The bell was ringing as the students entered. (Ongoing action)
    
- Our classes were starting later during the winter. (Temporary past situation)
    
- The fish was swimming in circles. (Ongoing past action)
    
- Lions were living near the village at that time. (Ongoing past state)
    

Other Sentences Rules :

- Negative Sentences
    

|   |
|---|
|Singular Subject (III) + was not + Verb (I) + ing + Object  <br>Other Subject + were not + Verb (I) + ing + Object  <br>I + was not + Verb (I) + ing + Object|

For sentences with Never:

|   |
|---|
|Singular Subject (III) + was never + Verb (I) + ing + Object  <br>Other Subject + were never + Verb (I) + ing + Object  <br>I + was never + Verb (I) + ing + Object|

- Interrogative Sentences 
    

|   |
|---|
|(W/H words) + Was + Singular Subject (III)/I + Verb (I) + ing + Object + ?  <br>(W/H words) + Were + Other Subject + Verb (I) + ing + Object + ?|

- Negative Interrogative Sentences
    

|   |
|---|
|(W/H words) + Was + Singular Subject (III)/I + not + Verb (I) + ing + Object + ?  <br>(W/H words) + Were + Other Subject + not + Verb (I) + ing + Object + ?|

## 7. Past Perfect Tense

  

The Past Perfect Tense is used to describe an action that was completed before another action or a specific time in the past (the "past of the past").

  

Base Rule (Affirmative Sentences) :

|   |
|---|
|All Subjects + had + Verb (III) + Object|

Examples:

- She had eaten the orange before lunch started. (Action before another past event)
    
- I had eaten my apple by the time you called. (Completed before a specific time)
    
- The Sun had risen before we woke up. (Completed before another action)
    
- The stars had appeared long before midnight. (Completed past action)
    
- The bell had rung before I reached the gate. (Action sequence)
    
- Our classes had started before the principal arrived. (Sequence of events)
    
- The fish had swum away before I could catch it. (Completed action)
    
- The lions had lived there for years before moving. (State completed before a past change)
    

Other Sentences Rules :

- Negative Sentences
    

|   |
|---|
|All Subjects + had not + Verb (III) + Object|

For sentences with Never:

|   |
|---|
|All Subjects + had never + Verb (III) + Object|

  

  

- Interrogative Sentences 
    

|   |
|---|
|(W/H words) + Had + All Subjects + Verb (III) + Object + ?|

- Negative Interrogative Sentences
    

|   |
|---|
|(W/H words) + Had + All Subjects + not + Verb (III) + Object + ?|

  

## 8. Past Perfect Continuous Tense

  

The Past Perfect Continuous Tense is used to show that an action started in the past and continued up until another point in the past. It emphasizes duration.

  

Base Rule (Affirmative Sentences) :

|   |
|---|
|All Subjects + had been + Verb (I) + ing + Object + (since/for) + Time|

Examples:

- She had been eating oranges for weeks to improve her health. (Duration of past action)
    
- I had been eating apples since morning before I felt full. (Continuous past action)
    
- The Sun had been rising behind the clouds for an hour before it cleared. (Past duration)
    
- The stars had been shining all night before the storm rolled in. (Continuous state)
    
- The bell had been ringing for a minute before someone answered. (Ongoing past action)
    
- Our classes had been starting early for months. (Past continuous habit)
    
- The fish had been swimming near the shore all day. (Continuous activity)
    
- Lions had been living in that cave for a decade before being discovered. (Long-term past duration)
    

Other Sentences Rules :

- Negative Sentences
    

  

|   |
|---|
|All Subjects + had not been + Verb (I) + ing + Object + (since/for) + Time|

For sentences with Never:

|   |
|---|
|All Subjects + had never been + Verb (I) + ing + Object + (since/for) + Time|

- Interrogative Sentences 
    

|   |
|---|
|(W/H words) + Had + All Subjects + been + Verb (I) + ing + Object + (since/for) + Time + ?|

- Negative Interrogative Sentences
    

|   |
|---|
|(W/H words) + Had + All Subjects + not been + Verb (I) + ing + Object + (since/for) + Time + ?|

## 9. Future Indefinite Tense

  

The Future Indefinite Tense (also known as the Simple Future) is used to express actions that will happen in the future, predictions, or sudden decisions.

  
  
  
  

Base Rule (Affirmative Sentences) :

|   |
|---|
|All Subject + will + Verb (I) + Object  <br>(Traditionally : I/We + shall + Verb (I) + Object)|

Examples:

- She will eat an orange tomorrow. (Future Action)
    
- I will eat an apple after lunch. (Decision/Plan)
    
- The Sun will rise at 6:15 AM tomorrow. (Future Fact)
    
- The stars will shine tonight if it is clear. (Prediction)
    
- The bell will ring in ten minutes. (Fixed Future Event)
    
- Our classes will start next Monday. (Scheduled Action)
    
- The fish will swim to the other side of the tank. (Future Action)
    
- Lions will live in peace in this sanctuary. (Future State)
